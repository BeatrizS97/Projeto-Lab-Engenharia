Objetivo

Objetivo Geral

  • Desenvolver uma plataforma intuitiva e responsiva para a criação, gestão e análise de formulários personalizados, permitindo que os usuários criem formulários dinâmicos com perguntas condicionais e obtenham relatórios detalhados das respostas.

Objetivos Específicos

  • Criar uma interface intuitiva e responsiva para todos os dispositivos.
  • Desenvolver um sistema para criação, edição e visualização de formulários.
  • Permitir categorização e definição de tipos de perguntas e respostas.
  • Implementar perguntas de múltipla escolha, texto livre e outros formatos.
  • Vincular perguntas a categorias e tipos para melhor organização.
  • Adicionar perguntas condicionais com base em respostas anteriores.
  • Criar fluxos dinâmicos com relações condicionais entre perguntas.
  • Facilitar a resposta aos formulários e armazenar dados de forma eficiente.
  • Garantir segurança e integridade dos dados com criptografia e autenticação.
  • Gerar relatórios dinâmicos com tabelas, gráficos e exportação (CSV/PDF).
  • Implementar filtros para análise de dados.
  • Criar um sistema de autenticação com perfis de Administrador e Usuário.
  • Assegurar um banco de dados escalável e de alto desempenho.
  • Garantir suporte a alta carga de usuários sem perda de performance.
  • Preparar a solução para hospedagem em nuvem com alta disponibilidade.
